■はじめに
【くすぐりエスケープ！２】をダウンローして頂き誠にありがとうございます。
本ゲームは脱出アクションゲームです。
ダンジョンを徘徊する魔物に敗北したり、罠にかかったりするとヒロインのカレンはくすぐり責めを味わうことになってしまいます。

プレイ時間は30分程度を目安としています。
妖精さんのいる部屋にあるクリスタルに話しかけると、全てのイベントを見ることのできる回想部屋にいつでも行けますので、本編クリアが面倒になった場合はそちらをご利用ください。


■ゲームの起動方法
『Game.exe』をダブルクリックで起動して下さい。


■操作説明
Aボタン：攻撃
決定ボタン（Z）：調べる
Xボタン：メニュー

・冒頭で手に入る「不思議なオーブ」をメニューから利用すると、妖精さんのいる休憩地点にワープすることができます。

・レベルアップの概念はありません。

・先に進めなくなったら同梱の「★ヒント.txt」を読んでください。


■ご注意
・当ゲームはRPGツクールMVによって制作しています。
・性的描写（くすぐり責めのみ）を含むため、18才未満の方はプレイしないでください。
・当ゲームのデータ複製・再配布、および素材の一部もしくは全部の流用を禁じます。
・本ゲームのプレイにより何らかの問題が発生しても、当方としては責任を取りかねます。
